# 📱 Smart Document Scanner (Web)

OpenCV.js 기반의 웹 문서 스캐너 애플리케이션입니다. 교육용 목적으로 제작되었으며, 브라우저에서 바로 동장하는 **문서 감지 및 처** 기능을 제공합니다.

## ✨ 주요 기능

### 1. 📄 스마트 문서 스캔
- **문서 자동 감지**: 이미지에서 문서 영역을 자동으로 찾아냅니다.
- **Perspective Transform**: 기울어진 문서를 평면으로 반듯하게 펴줍니다.

### 2. 🎨 다양한 이미지 필터
- **흑백/스캔 모드**: 문서를 깔끔한 흑백 스캔본처럼 변환합니다. (Threshold 조절 가능)
- **이미지 꾸미기**:
  - **윤곽선 오버레이**: 문서의 테두리를 강조합니다.
  - **스팟 컬러**: 특정 색상(빨강, 초록, 파랑)만 남기고 흑백으로 변환합니다.
  - **블러/샤프닝**: 이미지의 질감을 조절합니다.
  - **회전 및 대칭**: 자유로운 편집이 가능합니다.

### 3. 🖨️ 출력 최적화
- **다양한 용지 지원**: A4, A3, B3, Letter 등 표준 용지 사이즈로 결과물을 출력할 수 있습니다.
- **자동 리사이즈**: 선택한 용지 크기에 맞춰 이미지가 자동으로 조정되어 다운로드됩니다. (300DPI 기준)

## 🛠️ 기술 스택
- **Core**: HTML5, CSS3 (Tailwind CSS)
- **Logic**: Vanilla JavaScript, React (UI)
- **Computer Vision**: OpenCV.js

## 🚀 실행 방법
이 프로젝트는 정적 웹 페이지로 구성되어 있습니다.
1. `index.html` 파일을 브라우저에서 실행합니다. (`Live Server` 권장)
2. 이미지를 드래그 앤 드롭하거나 업로드하여 스캔을 시작합니다.
